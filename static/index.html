Help & Information

About NEXRAD Data

NEXRAD (Next Generation Radar) is a network of 160 Doppler weather radars operated by the National Weather Service. This application visualizes Level 3 NEXRAD data products.

How to Use

  1. Enter a NEXRAD file URL or use the default sample
  2. Click "Load Radar Data" to process the data
  3. Adjust visualization settings in the left panel
  4. Use the timeline controls to animate data (when available)
